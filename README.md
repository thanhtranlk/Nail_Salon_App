# firebase-login

## Project setup

1. Install packages
    ```
    yarn install
    ```

2. Copy your firebase configuration file into `src/myconfig.ts`

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
